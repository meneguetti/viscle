## <p align="left">Viscle (Visual Lifecycle)</p>

Viscle provides a visual lifecycle for PHP request.


## Installation

```sh
composer require meneguetti/viscle:dev-master
```

## Requirements

PHP >= 7.1

Xdebug >= php_xdebug-2.7.0alpha1-7.1
